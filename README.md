## ğŸš€ Panduan Instalasi (Step-by-Step)

### 1. Persiapan Database (MySQL)

1. Buka **phpMyAdmin**.
2. Buat database baru dengan nama `resto_db`.


3. Impor file `resto_db.sql` yang tersedia di root folder.



### 2. Konfigurasi Backend (PHP)

1. Letakkan folder `resto_api` ke dalam direktori `htdocs` (XAMPP/WAMP).


2. Sesuaikan kredensial database di file `resto_api/config/database.php`:


```php
private $username = "root";
private $password = "";

```



### 3. Konfigurasi App (Flutter)

1. Buka folder `resto_management`.
2. Jika menggunakan Emulator Android, pastikan `lib/config/api_config.dart` mengarah ke `10.0.2.2`.


3. Jika menggunakan Device Fisik, ganti dengan alamat IP lokal komputer Anda.


4. Jalankan perintah:
```bash
flutter pub get
flutter run

```
